# 都市システム仕様（City System Design）

## 概要
都市はストラテジーゲームの中心となる拠点であり、資源生産・装備作成・施設効果などを管理する。

---

# 都市規模

都市は4段階の規模を持つ。

|規模|概要|
|---|---|
|村|小規模集落|
|町|発展した拠点|
|都市|地域の中心都市|
|大都市|国家レベルの都市|

都市規模は以下に影響する

- 人口上限
- 建設可能施設数
- 支配範囲
- 生産能力

---

# 都市能力

都市には文明レベルとして以下の能力が存在する。

- 鍛冶場Lv
- 魔法Lv
- 信仰Lv

これらは施設・装備・都市効果に影響する。

## Lv上限方針

都市能力Lvは内部的には **Lv7まで存在**する想定とする。
ただし、当面の実装・バランス調整では **Lv4を最大値** として扱う。

### 方針

- データ定義上の最大値: Lv7
- 当面の実装上限: Lv4
- Lv5〜Lv7は将来拡張用として予約する

この方針により、後から上位都市・遺産・国家補正・特殊施設などで段階拡張しやすくする。

### 装備効果

- 光属性付与

- アンデッド汚染の除去

# 軍事能力

軍事は都市の戦力投射能力を表す都市能力とする。

## 基本方針

- 基本の生成ユニットは **1ユニット = 1キャラ** とする
- 軍事Lvを上げると **部隊ユニット** を生成可能にする
- これにより、序盤は個体戦力中心、後半は部隊運用へ発展する構造にする

## 軍事Lvの役割

| 軍事Lv | 解放内容 |
|---|---|
| 1 | 個体ユニットのみ |
| 2 | 小隊ユニット解放 |
| 3 | 中隊ユニット解放 |
| 4 | 軍団ユニット解放 |
| 5〜7 | 将来拡張 |

## ユニット構造

### 個体ユニット

- 1ユニット = 1キャラ
- 装備可能
- スキルあり
- 英雄 / 騎士 / 魔術師 / 冒険者など

### 部隊ユニット

- 軍事Lvで解放される
- 抽象部隊として扱う
- 人数は内部で厳密管理しない
- 歩兵隊 / 弓兵隊 / 騎兵隊などの形で表現する

## 隊ユニット仕様

軍事Lv2以降で解放される **隊ユニット** には以下の戦闘補正を与える。

### 基本補正

- **HP倍率補正**（個体ユニットより高い耐久）
- **攻撃回数増加**（人数による攻撃回数の表現）

例（イメージ）

| ユニット種別 | HP補正 | 攻撃回数 |
|---|---|---|
| 個体ユニット | ×1 | 1回 |
| 隊ユニット | ×3〜×5 | 2〜3回 |
| 中隊ユニット | ×8〜×12 | 3〜5回 |

### 設計意図

- 隊ユニットは「多数の兵士」を **HPと攻撃回数で表現**する
- スキルや装備管理を増やさずに戦力を拡張できる
- 戦闘処理を軽く保ちながら軍隊規模の戦闘を表現する

---

## 設計意図

- 序盤は個体ユニット中心で運用する
- 中盤以降は軍事Lvにより部隊を展開できる
- ネームドや英雄が部隊指揮役になる構造へ拡張しやすくする

# 経済能力

経済は都市の資源生産・交易・収入効率を表す都市能力とする。

## 主な役割

- 資源生産量の増加
- 交易効率の上昇
- 金収入の増加
- 市場 / 商業施設の強化

## 経済Lvのイメージ

| 経済Lv | 主な効果 |
|---|---|
| 1 | 基本市場 |
| 2 | 資源生産補正 |
| 3 | 交易路強化 |
| 4 | 商業都市化 |
| 5〜7 | 将来拡張 |

## 関連施設例

- 市場
- 商人ギルド
- 倉庫
- 港

## 設計意図

- 経済都市は装備素材・交易・資金面を支える
- 鍛冶 / 魔法 / 信仰の基盤となる資源供給拠点にする
- 少都市型（1〜8都市）のため、都市ごとの専門化を明確にする

# 都市設計の目的

都市能力によって

- 軍事都市
- 魔法都市
- 宗教都市

など都市の個性を生み出す。

これにより

- 国家戦略
- 装備生産
- 領土運営

が分化する。



## 装備レアリティ段階

装備には都市発展Lvに対応したレアリティ段階を設定する。

- Lv1 : 一般
- Lv2 : 上質
- Lv3 : 高品質
- Lv4 : 希少
- Lv5 : 遺物
- Lv6 : 伝説
- Lv7 : 神器

このレアリティは主に以下に影響する。
- 基礎性能倍率
- 付与できる魔法数
- 特殊効果の強さ

## 研究ツリー基本ルール

都市能力（鍛冶 / 魔法 / 信仰）は研究ツリー方式で成長する。

基本仕様:
- 最大定義Lvは7だが、当面の実装上限はLv4
- 分岐は **Lv2から開始** する

研究ツリー構造（基本）

Lv1
 ↓
Lv2
 ↓ 分岐開始
 ├─ 系統A
 └─ 系統B

Lv3以降は選択した系統に沿って強化される。

この設計により
- 都市の専門化
- プレイごとの戦略差
- 都市ごとの役割分担

が発生する。

## 魔法Lv / 信仰Lv 段階


魔法Lvと信仰Lvは同じ段階構造を持つ。

| Lv | 段階 |
|---|---|
| 1 | 初級 |
| 2 | 下級 |
| 3 | 中級 |
| 4 | 上級 |
| 5 | 下級神 |
| 6 | 中級神 |
| 7 | 上級神 |

この段階は以下に影響する。
- 付与できる魔法の強さ
- 建設できる魔法施設
- 信仰系効果（浄化・アンデッド抑制・精神耐性など）

## 装備性能倍率ルール



装備の基礎性能は都市の発展Lvに応じて強化される。

基本方針:
- Lv1 : 基準値 (×1.0)
- Lv2 : ×1.2
- Lv3 : ×1.5
- Lv4 : ×2.0

拡張予定Lv:
- Lv5 : ×2.3
- Lv6 : ×2.6
- Lv7 : ×3.0

この倍率は主に以下へ適用される。
- 武器の威力
- 防具の防御値
- 付与される魔法効果の基礎値

例:
- 基礎威力 20 の武器
  - Lv1都市 → 威力20
  - Lv4都市 → 威力40
  - Lv7都市 → 威力60

この倍率は**鍛冶Lv・魔法Lv・信仰Lvによる付与効果の計算の基礎値**として扱う。
