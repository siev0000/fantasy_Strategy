# マップ画面 概要仕様（2026-03-05）

## 目的
- ワールドラップ外周コピーを維持したまま、表示基準を常に中央にする。
- 最大縮小時に表示できる範囲を「作成マップの 1.20 倍まで」に制限する。
- ドラッグで過度に余白へ流れないようにする。

## 画面基準
- 描画基準サイズ: `1280x720`
- マップ初期表示: マップ中心を画面中心に配置
- ワールドラップ: `ON` 時は外周コピーを表示（上下左右）

## ズーム仕様
- 最小ズーム（最大縮小）:
  - 表示範囲がマップの `1.20倍` を超えない値を自動計算
  - 最小ズーム時も中心はマップ中央
- 最大ズーム（最大拡大）: `400%`
- ホイール操作: 倍率のみ変更（ポインタ追従ズームは使わない）

## ドラッグ仕様
- ドラッグ時のカメラ制限は「中央基準の範囲制限」で行う。
- ラップON時の移動許容:
  - 中央位置から、マップ幅/高さの `±10%`（= 1.20倍の半分余白）まで
- ラップOFF時:
  - 通常境界内に制限
  - 表示窓がマップより大きい場合は中央固定

## 期待される見え方
- 最大縮小時:
  - 画面全体が左上寄せにならない
  - 下/右が欠けない
  - 必要な外周コピーは表示される
- 中間/拡大時:
  - ドラッグで大きく逸脱しない
  - スクロールごとに位置が飛ばない
