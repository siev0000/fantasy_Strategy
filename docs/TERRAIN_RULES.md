# 地形生成ルールメモ

## 目的
- 島マップを安定生成し、プレイしやすい平地ベースを維持しつつ、山・丘・森・川の地理的な納得感を出す。

## 現在の実装フロー
1. 全マスを `海` で初期化  
2. `島形状パターン` を使って陸地を生成  
3. 陸地を一度すべて `平地` に統一  
4. 山クラスタを生成（最低3塊）  
5. 丘クラスタを生成（最低2塊、山周辺優先）  
6. 森クラスタを生成  
7. 砂漠クラスタを生成（最低6塊、縦中央帯バイアス）  
8. 孤立補正（孤立山→丘、孤立森→平地）  
9. 地形まとまり補正（塊に寄せる同化）  
10. 湖を条件生成（砂漠中央オアシスを優先）  
11. 川を生成（地形上への重ね表示）

## 島形状パターン
- `標準諸島 (balanced)`
- `大陸型 (continent)`
- `多島海 (archipelago)`
- `双子島 (twins)`
- `列島型 (chain)`

## 地形ごとの主ルール

### 平地
- 島内の基準地形。まず陸地を平地化してから他地形を上書きする。

### 山
- 総量目標からクラスタサイズをランダム分配して生成。
- 最低クラスタ数は `3`。
- 海沿いを避け、内陸寄りに生成。

### 丘
- 総量目標からクラスタサイズをランダム分配して生成。
- 最低クラスタ数は `2`。
- 山の隣接マスを優先して生成（山麓帯）。

### 森
- 平地/丘を対象にクラスタ生成。
- 山周辺が過密な場所は優先度を下げる。

### 砂漠
- 陸地の `35%` を目標に生成。
- 最低クラスタ数は `6`。
- 海に隣接しない平地を中心に生成。
- 縦方向バイアスあり:
- 画面中央（上下の中間帯）で高確率
- 上端/下端へ行くほど低確率

### 湖
- 後処理のみで生成（直接候補には入れない）。
- 条件:
- 山岳が2マス以上隣接（山麓）
- または森が4マス以上隣接かつ海隣接なし（森の中心）
- 上限: マップ総マスの `5%`
- 追加:
- 砂漠タイル群の重心（砂漠中央付近）に `オアシス湖` を1つ優先配置

### 川
- 山岳を起点に海/湖へ向かう流路を作る。
- `河川` タイルに置換せず、`川オーバーレイ` として地形に重ねる。
- 例: `山 + 川`, `丘 + 川`, `平地 + 川`

## 表示仕様
- ヘックスは `pointy-top`（角が頂点）  
- 行オフセット方式で噛み合わせ  
- `mapGrid` 内で自動スケーリング表示

## 主要調整パラメータ（現行）
- 島の陸地率: パターンごとに `landMin / landMax`
- 山割合目安: 陸地の約 `14%`
- 丘割合目安: 陸地の約 `16%`
- 森割合目安: 陸地の約 `24%`
- 砂漠割合目安: 陸地の `35%`（最低6クラスタ）
- 湖上限率: 総マスの `5%`
