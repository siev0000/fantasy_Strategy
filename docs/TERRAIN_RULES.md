# 地形生成ルールメモ

## 目的
- 島マップを安定生成し、プレイしやすい平地ベースを維持しつつ、山・丘・森・川の地理的な納得感を出す。

## 現在の実装フロー
1. 全マスを `海` で初期化  
2. `島形状パターン` を使って陸地を生成  
3. 高度マップを生成（島中央ほど高く、海沿いほど低くなりやすい）  
4. 高度順位で `山岳` / `丘陵` / `平地` を割り当て  
5. 森クラスタを生成  
6. 砂漠クラスタを生成（最低6塊、縦中央帯バイアス）  
7. 孤立補正（孤立山→丘、孤立森→平地）  
8. 地形まとまり補正（塊に寄せる同化）  
9. 湖を条件生成（砂漠中央オアシスを優先）  
10. 川を生成（地形上への重ね表示）

## 島形状パターン
- `標準諸島 (balanced)`
- `リアル島 (realistic)`
- `大陸型 (continent)`
- `多島海 (archipelago)`
- `双子島 (twins)`
- `列島型 (chain)`

## 地形ごとの主ルール

### 平地
- 島内の基準地形。まず陸地を平地化してから他地形を上書きする。

### 山
- 高度が高いマスから優先して割り当てる。
- 島中央ほど高度が高くなりやすいため、山岳は中央寄りに出やすい。

### 丘
- 山岳に次いで高度の高いマスへ割り当てる。
- 中央高地の周辺帯に形成されやすい。

### 森
- 平地/丘を対象にクラスタ生成。
- 山周辺が過密な場所は優先度を下げる。

### 砂漠
- 陸地の `35%` を目標に生成。
- 最低クラスタ数は `6`。
- 海に隣接しない平地を中心に生成。
- 縦方向バイアスあり:
- 画面中央（上下の中間帯）で高確率
- 上端/下端へ行くほど低確率

### 湖
- 後処理のみで生成（直接候補には入れない）。
- 条件:
- 山岳が2マス以上隣接（山麓）
- または森が4マス以上隣接かつ海隣接なし（森の中心）
- 上限: マップ総マスの `3%`
- 追加:
- 砂漠タイル群の重心（砂漠中央付近）に `オアシス湖` を1つ優先配置
- オアシス条件:
- 候補マスの周囲6近傍のうち、5以上が砂漠であること

### 川
- 山岳を起点に海/湖へ向かう流路を作る。
- `河川` タイルに置換せず、`川オーバーレイ` として地形に重ねる。
- 例: `山 + 川`, `丘 + 川`, `平地 + 川`
- 流速仕様（暫定）:
- `流速 = 基本流速(1) × 勾配補正(高度差) × 地質係数 × (1 - 森林率×0.4)`
- 川計算のため、全マスが `高度` / `地質係数` / `森林率` を持つ前提とする。

## 表示仕様
- ヘックスは `pointy-top`（角が頂点）  
- 行オフセット方式で噛み合わせ  
- `mapGrid` 内で自動スケーリング表示

## 主要調整パラメータ（現行）
- 島の陸地率: パターンごとに `landMin / landMax`
- 山割合目安: 陸地の約 `14%`
- 丘割合目安: 陸地の約 `16%`
- 森割合目安: 陸地の約 `24%`
- 砂漠割合目安: 陸地の `35%`（最低6クラスタ）
- 湖上限率: 島面積（陸地）の `3%`
- 高度生成: 中央隆起 + 海岸減衰 + ノイズ + 平滑化
