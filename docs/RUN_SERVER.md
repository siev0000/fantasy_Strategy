# 起動手順（マルチプレイ対応）

## 前提
- Node.js 18 以上

## セットアップ
```bash
npm install
```

## 起動
```bash
npm start
```

同じPCなら `http://localhost:3000` にアクセスします。  
別PCから参加する場合は、サーバーPCのIPで `http://<サーバーIP>:3000` にアクセスします。

## 複数人プレイ
1. 全員が同じサーバーURLへアクセスする
2. 1人が「ルーム作成」を押してルームIDを共有する
3. 他プレイヤーは共有されたルームIDを入力して「ルーム参加」を押す
4. 誰かの操作（攻撃/スキル/ターン終了/リセット）が全員に同期される

## 開発起動（自動再起動 + ブラウザ自動リロード）
```bash
npm run dev
```

`web/` `assets/` `config/` `data/` 配下の変更を保存すると、接続中ブラウザが自動で再読み込みされます。
