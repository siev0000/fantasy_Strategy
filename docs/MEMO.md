# ファンタジーストラテジーゲーム 概要メモ

## コンセプト
- Civベースの国家運営ゲーム。
- 外交・資源・都市開発を主軸に進行する。
- ファンタジー要素（種族・災害・遺産）を国家運営に統合する。

## 登場種族
- 只人
- エルフ
- オーガ
- ゴブリン
- 竜人
- 悪魔
- 天使
- ヴァンパイア

## コアループ（Civベース）
- 1. 国家設定（国家体制 / 外交姿勢 / 国是 / 派閥特性）
- 2. 都市設定（通常枠の都市特性を選択）
- 3. 内政（資源生産・施設建設・社会資源管理）
- 4. 外交（交易・交換・条約）
- 5. イベント処理（時代イベント / 災害）
- 6. 次ターンへ

## 最初に決める項目
- 主要種族
- 国家体制

## 都市特性の決定ルール
- 都市特性は開始時に固定選択しない。
- 町を成長させて「都市化」するタイミングで決定する。
- 都市化時点の数値（例: 生産寄り / 交易寄り / 信仰寄り / 軍事寄り）に応じて、対応する都市特性を付与する。
- これにより、プレイヤーの運用結果が都市特性に反映される設計とする。

## 土地と開拓ルール
- 土地タイプ（例: 森 / 丘 / 平地 / 河川 / 沼地）ごとに、取得できる資源テーブルを持つ。
- 土地を確保した後、開拓方針を選んで産出資源を確定する。
- 開拓方針の例: 前線拠点化 / 資源地化 / 防衛地化 / 都市化候補地。
- 開拓により、資源産出・治安・防衛力・都市化条件の進行が変化する。

## 土地開拓の段階仕様
- 土地タイプごとに基礎産出を設定する（森=木材寄り、丘=石材/鉱物寄り、平地=食料寄り、など）。
- 各土地に開拓度（0〜100）を持たせる。
- 開拓行動で開拓度が上昇し、災害や放置で低下する。
- 開拓できる土地は「既に占拠済みの土地に隣接するマス」のみとする。
- 開拓度50以上で畑化可能。
- 開拓度80以上で町化可能（追加条件: 治安・人口が基準値以上）。
- 開拓度100で開拓完了扱いとし、物資取得率に恒久ボーナスを付与する。
- 産出計算の方針: 最終産出 = 基礎産出 × (1 + 開拓度ボーナス + 施設ボーナス)。

## 規模ごとの管理可能区画数（確定）
- 村: 3区画
- 町: 6区画
- 都市: 10区画

## 規模ごとの人口上限（確定）
- 村: 人口10〜39（上限40）
- 町: 人口40〜119（上限120）
- 都市: 人口120〜299（上限300）
- 人口は抽象値として扱い、人口1ポイント ≒ 現実100人で換算する。

## 開拓コスト方針
- 開拓は1人でも実行可能とする。
- 参加人数が少ないほど必要ターン数は増える。
- キャラLvと参加人数で開拓速度が変化する。
- 例: Lv10で4ターン、Lv5で8ターン、Lv10を4人投入で1ターン。
- 正式な計算式（Lv係数・人数係数）は後で確定する。

## 必要資源ルール
- 維持・行動で常時必要な資源は食料のみ。
- 建物建設時にのみ、対応する建設資源（木材・石材・鉱物など）を消費する。

## 採用するコア要素
- 国家体制
- 外交姿勢
- 国是
- 派閥特性
- 都市特性（通常枠・レア枠・遺産枠）
- 資源（食料 / 一般 / 戦略 / 貴重）
- 社会資源（人口 / 文化 / 信仰力 / 統治力）
- 地形
- 天候
- 災害（都市災害 / 天候災害 / モンスター災害）
- 技術分野一覧
- 施設カテゴリ
- 交易可能項目
- 市民不満度段階
- 時代イベント

## プレイヤーが設定できること
- 新規ゲーム開始時
- 種族
- 国家体制
- 外交姿勢
- 国是
- 派閥特性（任意）
- 都市化時の都市特性判定ルール（初期設定）
- ターン中
- 外交姿勢の方針転換
- 国是の再設定
- 都市運用方針の切り替え
- 交易提案（資源・条約）
- 条件達成時
- 都市特性レア枠の解放
- 都市遺産枠の建設
- 災害対策の実行

## 国データ/キャラデータ設計
- 国とキャラの保持項目、および影響先は `config/entity_data_schema.json` を基準にする。
- セーブデータ（保持）と計算ロジック（内政/外交/イベント）を分離する。

## 保留・削除（現時点）
- 補給線状態（初期MVPから除外）
- 指揮官特性（初期MVPから除外）
- クエスト・事件（災害と時代イベントに統合）
- 勝利条件ロジック（定義のみ、実装は後回し）

## タスクメモ
- 未確定ルールの管理は `docs/RULE_TASKS.md` を参照する。

## これから決めること（タスク）
- 1. ゲーム開始時の設定項目を確定する
- 国家体制 / 外交姿勢 / 国是 / 派閥特性 / 種族 の必須・任意を確定
- 2. 都市の基本役割を確定する
- 生産都市 / 軍事都市 / 交易都市 / 信仰都市 の役割差を数値で確定
- 3. 施設カテゴリごとの建設条件を確定する
- 施設カテゴリごとに必要資源と解放条件を決める
- 4. 資源収支の基準式を確定する
- 食料・一般・戦略・貴重資源の毎ターン増減式を決める
- 5. 市民不満度の増減条件を確定する
- 食料収支・災害・政策で不満がどう上下するかを決める
- 6. 外交交換のルールを確定する
- 交換可能項目・交換レート・外交姿勢による補正を決める
- 7. 災害の発生ルールを確定する
- 都市災害 / 天候災害 / モンスター災害の発生率と継続ターンを決める
- 8. 時代イベントの効果を確定する
- 各イベントのメリット・デメリットと発生条件を決める
- 9. 種族ごとの基礎ステータスを確定する
- HP / ATK / DEF / MAG / RES / SPD / 基礎移動 と得意技術分野を決める
- 10. MVP範囲を固定する
- 先に実装する対象を「内政・外交・災害」の3本に限定するか確定
